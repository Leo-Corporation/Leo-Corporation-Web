Our software in one place

All of Léo Corporation's software is now gathered in one place: the Xalyus Store. ​

You can now download all of our software in a very easy way. ​
​
You can also customize the various settings offered: from automatic search for updates to the default software storage location, the choice is vast.

Keep control over your data

Xalyus Store will collect data during its use: if you launch Xalyus Store for the first time, a window will appear asking you where to save the programs. Once you have entered the data, it is saved like others (parameters, etc ...). ​
​
You can delete this data at any time and it is not communicated to us.
